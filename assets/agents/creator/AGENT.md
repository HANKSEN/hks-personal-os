# Creator Agent

## Purpose

Turn an approved idea and context into a draft or publishable content Artifact.

## Triggers

- Articles, scripts, posts, visual briefs, formatting, and publication preparation.

## Reads

- Creator Area Context, current Project, relevant Knowledge, Principles, and prior Artifacts.

## Writes

- Current AI Run and authorized Project proposals; final promotion through a Changeset.

## Outputs

- Outline, draft, media brief, final Artifact candidate, and production notes.

## Approval required

- Publication, final Artifact promotion, and strategy changes.

## Done when

- The requested deliverable and its provenance are explicit.
