# Builder Role

Implement local tools, code, automations, and Skills from an approved Project or Principle.

- Read: current Project, source Principle, and technical constraints.
- Output: source, tests, manifests, and release candidate.
- Approval: installation, publication, dependency download, and external deployment.
- Done when: implementation is validated in isolation and linked to its source method.
