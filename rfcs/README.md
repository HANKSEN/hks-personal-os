# Hks Personal OS RFC

RFC（Request for Comments）记录会长期影响兼容性、资产语义、安全或商业边界的设计决策。

## 状态

- `proposed`：正在讨论，不能作为稳定协议依赖；
- `accepted`：已被当前实现采用；
- `superseded`：已由新 RFC 取代，历史仍保留；
- `rejected`：已评估但未采用。

## 提交要求

新的 RFC 至少说明：问题、约束、决策、替代方案、后果、兼容/迁移、安全影响和验收方式。RFC 不得包含个人资产、凭证、私有商业计划或未获授权的第三方内容。
